# Chess アピールページ 静的サイト版

React / Next.js を使わない、HTML / CSS / JavaScript だけのアピールサイトです。

## ファイル構成

```text
index.html      アピールページ本体
readme.html     READMEボタン用の仮ページ
style.css       デザイン
script.js       スマホメニューなどの軽い処理
```

## 動作

- `index.html` をブラウザで開くとアピールページが表示されます。
- 「サービスを見る」は、すでにデプロイ済みの `https://chess-nine-xi.vercel.app/` に遷移します。
- 「READMEを見る」は、同じフォルダ内の `readme.html` に遷移します。

## 写真の差し替え

ページ内の「写真」と書いてある枠を、後から実際のスクリーンショットに差し替えてください。
