Start

利用開始の流れ

初めて使う場合は、管理者が先に組織、従業員、シフト枠を準備します。その後、従業員が希望シフトを提出し、管理者が承認します。

  1. 管理者がアカウントを新規登録します。
  2. 管理者がメール確認を完了します。
  3. 管理者がログインします。
  4. 管理者が組織を作成します。
  5. 管理者が従業員を登録します。
  6. 管理者がシフト枠を作成します。
  7. 従業員が組織IDとメールアドレスでログインします。
  8. 従業員が希望シフトを追加して送信します。
  9. 管理者が提出された希望シフトを確認します。
  10. 管理者が希望シフトを承認します。
  11. 従業員が承認済みシフトを確認します。

従業員に共有する必要がある情報は、管理者ホームに表示される6桁の組織IDと、従業員登録で登録したメールアドレスです。

Admin

管理者の操作手順

管理者は、組織、従業員、シフト枠、希望シフト、勤務時間、給与設定を管理します。

1. 管理者アカウントを新規登録する

操作手順

  1. サービスのURLを開きます。
  2. 利用者選択画面で「管理者」を選択します。
  3. 管理者ログイン画面で「新規登録」を押します。
  4. メールアドレスを入力します。
  5. パスワードを入力します。
  6. 「登録」を押します。
  7. 入力したメールアドレスに届く確認メールを開きます。
  8. メール内の確認リンクを開きます。
  9. メール確認を完了します。
  10. 管理者ログイン画面に戻ります。

メール確認が完了していない場合、管理者としてログインできません。パスワードは6文字以上で入力してください。

2. 管理者としてログインする

操作手順

  1. サービスのURLを開きます。
  2. 利用者選択画面で「管理者」を選択します。
  3. メールアドレスを入力します。
  4. パスワードを入力します。
  5. 「次へ」を押します。
  6. 組織選択画面へ移動します。

ログインできない場合

  • メールアドレスが間違っていないか確認します。
  • パスワードが間違っていないか確認します。
  • メール確認が完了しているか確認します。
  • 新規登録したアカウントでログインしているか確認します。

3. 組織を追加・選択する

組織を追加する手順

  1. 管理者としてログインします。
  2. 組織選択画面で「組織を追加」を押します。
  3. 組織名を入力します。
  4. 必要に応じて部署・拠点名を入力します。
  5. 「登録」を押します。
  6. 組織が作成され、管理者ホームへ移動します。

組織を選択する手順

  1. 管理者としてログインします。
  2. 組織選択画面を確認します。
  3. 管理したい組織を選択します。
  4. 「次へ」を押します。
  5. 選択した組織の管理者ホームへ移動します。

組織を作成すると、従業員ログインに使う6桁の組織IDが自動発行されます。従業員には「組織ID」と「登録済みメールアドレス」を共有してください。

4. 従業員を登録する

操作手順

  1. 管理者ホームを開きます。
  2. 「従業員登録」を押します。
  3. 姓を入力します。
  4. 名を入力します。
  5. メールアドレスを入力します。
  6. 雇用形態を選択します。
  7. 「登録」を押します。
  8. 登録完了メッセージを確認します。
  9. 発行された従業員IDを確認します。
姓従業員の名字を入力します。
名従業員の名前を入力します。
メールアドレス従業員ログインに使用するメールアドレスです。
雇用形態正社員、アルバイト、契約社員などから選択します。

同じメールアドレスを重複して登録することはできません。従業員はここで登録されたメールアドレスを使ってログインします。

5. 従業員情報を編集・削除する

編集手順

  1. 管理者ホームを開きます。
  2. 「従業員登録」を押します。
  3. 登録済み従業員一覧を確認します。
  4. 編集したい従業員の編集ボタンを押します。
  5. 姓、名、メールアドレス、雇用形態を変更します。
  6. 保存します。
  7. 更新完了メッセージを確認します。

従業員情報を更新すると、その従業員に紐づく希望シフトの従業員名、メールアドレス、雇用形態も更新されます。

削除手順

  1. 登録済み従業員一覧を確認します。
  2. 削除したい従業員のゴミ箱ボタンを押します。
  3. 確認画面を読みます。
  4. 削除を実行します。
  5. 削除完了メッセージを確認します。

従業員を削除すると、その従業員に紐づく希望シフトも削除されます。削除後は元に戻せないため、実行前に確認してください。

6. シフト枠を追加する

操作手順

  1. 管理者ホームを開きます。
  2. 「シフト管理」を押します。
  3. 「シフト枠を追加」を押します。
  4. 日付を入力します。
  5. 開始時刻を入力します。
  6. 終了時刻を入力します。
  7. 募集人数を入力します。
  8. 保存します。
  9. シフト管理画面に追加されたことを確認します。
日付募集するシフトの日付です。
開始時刻勤務開始時刻です。
終了時刻勤務終了時刻です。
募集人数そのシフトで承認できる人数です。

開始時刻と終了時刻を同じ時刻にすることはできません。終了時刻が開始時刻より早い場合は、翌日の終了時刻として扱われます。

7. シフト枠を編集・削除する

編集手順

  1. 管理者ホームを開きます。
  2. 「シフト管理」を押します。
  3. 編集したいシフト枠を探します。
  4. 鉛筆ボタンを押します。
  5. 内容を変更します。
  6. 保存します。
希望者がいない場合日付、開始時刻、終了時刻、募集人数を編集できます。
希望者がいる場合募集人数のみ変更できます。日付、開始時刻、終了時刻は変更できません。

削除手順

  1. 削除したいシフト枠を探します。
  2. ゴミ箱ボタンを押します。
  3. 確認画面を読みます。
  4. 削除を実行します。

シフト枠を削除すると、そのシフト枠に紐づく希望シフトも削除されます。

8. 希望シフトを確認・承認する

希望シフトを確認する手順

  1. 管理者ホームを開きます。
  2. 「シフト管理」を押します。
  3. 日付ごとのシフト枠を確認します。
  4. 各シフト枠の希望者を確認します。
  5. 「承認待ち」に表示されている従業員を確認します。

承認する手順

  1. 承認したいシフト枠を探します。
  2. 「承認待ち」の従業員を確認します。
  3. 承認したい従業員の「承認」を押します。
  4. 対象の従業員が「承認済み」に移動したことを確認します。

承認済みの人数が募集人数に達している場合、それ以上は承認できません。

承認を取り消す手順

  1. 承認を取り消したいシフト枠を探します。
  2. 「承認済み」の従業員を確認します。
  3. 対象従業員のゴミ箱ボタンを押します。
  4. 確認画面を読みます。
  5. 実行します。
  6. 対象の希望が「承認待ち」に戻ったことを確認します。

承認済みの希望に対してゴミ箱ボタンを押した場合、希望シフト自体を削除するのではなく、承認を取り消します。

9. おすすめ承認組み合わせを確認する

操作手順

  1. 従業員が「一緒に働きやすさ設定」を保存します。
  2. 管理者が「シフト管理」を開きます。
  3. 希望者が複数いるシフト枠を確認します。
  4. 相性スコアを確認します。
  5. 「おすすめ承認組み合わせ」を確認します。
  6. 問題なければ「おすすめを一括承認」を押します。

おすすめ承認組み合わせは、募集人数に対して相性スコアが高い組み合わせを表示します。最終的に承認するかどうかは管理者が判断します。

10. 従業員シフト表を確認する

操作手順

  1. 管理者ホームを開きます。
  2. 「従業員シフト表」を押します。
  3. 従業員一覧を確認します。
  4. 確認したい従業員を選択します。
  5. その従業員のシフト希望を確認します。

検索欄では、従業員名、メールアドレス、従業員IDで絞り込めます。

11. 勤務時間を確認し、給与設定を変更する

勤務時間を確認する手順

  1. 管理者ホームを開きます。
  2. 「勤務時間」を押します。
  3. 対象月を選択します。
  4. 総勤務時間を確認します。
  5. 従業員ごとの勤務時間を確認します。
  6. 月ごとの給与目安を確認します。
  7. 年間給与目安を確認します。

勤務時間と給与は、承認済みかつ終了済みのシフトをもとに計算されます。承認待ちのシフトや、まだ終了していないシフトは集計されません。

給与設定を変更する手順

  1. 勤務時間画面を開きます。
  2. 給与設定エリアを確認します。
  3. 雇用形態ごとの時給を入力します。
  4. 深夜開始時刻を入力します。
  5. 深夜終了時刻を入力します。
  6. 深夜倍率を入力します。
  7. 保存します。
時給雇用形態ごとの1時間あたりの給与です。
深夜開始時刻深夜手当を開始する時刻です。
深夜終了時刻深夜手当を終了する時刻です。
深夜倍率深夜時間にかける倍率です。

12. 組織を削除する

操作手順

  1. 管理者ホームを開きます。
  2. 組織削除ボタンを押します。
  3. 確認画面を読みます。
  4. 削除を実行します。

組織を削除すると、従業員、シフト枠、希望シフト、働きやすさ設定、給与設定が削除されます。

削除後は元に戻せません。本当に削除してよい組織か確認してから実行してください。

Employee

従業員の操作手順

従業員は、管理者が登録した組織IDとメールアドレスを使ってログインし、希望シフトの提出やシフト確認を行います。

1. 従業員としてログインする

操作手順

  1. サービスのURLを開きます。
  2. 利用者選択画面で「従業員」を選択します。
  3. 組織IDを入力します。
  4. メールアドレスを入力します。
  5. 「次へ」を押します。
  6. 確認用コード画面へ移動します。
  7. 確認用コードを入力します。
  8. 「次へ」を押します。
  9. 従業員ホームへ移動します。
組織ID管理者ホームに表示される6桁の数字です。
メールアドレス管理者が従業員登録で登録したメールアドレスです。

2. 従業員ホームを確認する

従業員ホームでは、自分に関するシフト情報を確認できます。

  • 自分の名前
  • 所属組織
  • 雇用形態
  • 直近の確定シフト
  • 今月の勤務時間
  • 年間の勤務時間
  • 今月の給与目安
  • 年間の給与目安
  • 承認待ちの希望シフト
  • 承認済みのシフト

3. 希望する日付とシフト枠を選択する

希望シフト入力画面を開く

  1. 従業員ホームを開きます。
  2. 「希望シフト入力」を押します。
  3. 希望シフト入力画面へ移動します。

日付を選択する

  1. カレンダーを確認します。
  2. 希望を出したい日付を探します。
  3. 選択できる日付を押します。
  4. 選択した日付のシフト枠を確認します。

シフト枠を選択する

  1. シフト枠のプルダウンを開きます。
  2. 希望する時間帯を選択します。
  3. 募集人数と希望人数を確認します。

管理者がシフト枠を作成していない日、開始済みのシフト、終了済みのシフト、すでに自分が希望を出しているシフトしかない日は選択できません。

4. 希望を追加して送信する

希望を追加する手順

  1. シフト枠を選択します。
  2. 「希望を追加」を押します。
  3. 右側の「追加したシフト希望」に表示されることを確認します。

この時点では、まだ管理者には送信されていません。送信前であれば、追加した希望を削除できます。

追加した希望を削除する手順

  1. 「追加したシフト希望」を確認します。
  2. 削除したい希望シフトを探します。
  3. 「削除」を押します。
  4. 一覧から消えたことを確認します。

希望シフトを送信する手順

  1. 追加したシフト希望を確認します。
  2. 日付と時間に間違いがないか確認します。
  3. 「シフト希望を送信」を押します。
  4. 確認画面を読みます。
  5. 「送信する」を押します。
  6. 送信が完了します。
  7. 従業員ホームに戻ります。
  8. シフト希望一覧に表示されることを確認します。

送信後は、従業員側から変更できません。変更したい場合は、管理者に連絡してください。

5. 承認待ち・承認済みシフトを確認する

承認待ちシフトを確認する手順

  1. 従業員ホームを開きます。
  2. 「シフト希望一覧」を確認します。
  3. 「承認待ち」を確認します。

承認待ちのシフトは、まだ確定していません。管理者が承認すると、承認済みに移動します。

承認済みシフトを確認する手順

  1. 従業員ホームを開きます。
  2. 「シフト希望一覧」を確認します。
  3. 「承認済み」を確認します。

承認済みのシフトは、管理者が承認した確定シフトです。

6. 直近のシフト・勤務時間・給与目安を確認する

直近のシフトを確認する手順

  1. 従業員ホームを開きます。
  2. 「直近のシフト」を確認します。
  3. 日付、時間、給与目安、ステータスを確認します。

直近に承認済みシフトがない場合は、「直近のシフトはありません」と表示されます。

勤務時間を確認する手順

  1. 従業員ホームを開きます。
  2. 「今月の勤務時間」を確認します。
  3. 「年間の勤務時間」を確認します。

給与目安を確認する手順

  1. 従業員ホームを開きます。
  2. 「今月のお給料」を確認します。
  3. 「年間のお給料」を確認します。

勤務時間と給与は、承認済みかつ終了済みのシフトをもとに計算されます。実際の給与額と完全に一致するとは限りません。

7. 一緒に働きやすさ設定を入力する

操作手順

  1. 従業員ホームを開きます。
  2. 「一緒に働きやすさ設定」を押します。
  3. 他の従業員一覧を確認します。
  4. 各従業員ごとにスライダーを動かします。
  5. -5から+5の範囲で設定します。
  6. 全員分を確認します。
  7. 「保存する」を押します。
  8. 保存完了メッセージを確認します。
+5とても一緒に働きやすい。
0普通。
-5一緒に働きにくい。

未入力の相手は0点として保存されます。この設定は、管理者側の相性スコアやおすすめ承認組み合わせに使われます。

8. ログアウトする

操作手順

  1. 従業員ホームを開きます。
  2. 右上のログアウトボタンを押します。
  3. ログイン画面へ戻ります。

共有端末を使っている場合は、利用後に必ずログアウトしてください。

FAQ

よくある困りごと

管理者でログインできない

  • メールアドレスが正しいか確認します。
  • パスワードが正しいか確認します。
  • メール確認が完了しているか確認します。
  • 登録したアカウントでログインしているか確認します。

従業員でログインできない

  • 組織IDが6桁の数字になっているか確認します。
  • 組織IDが正しいか確認します。
  • メールアドレスが正しいか確認します。
  • 管理者がそのメールアドレスを登録しているか確認します。
  • 別の組織IDを入力していないか確認します。

希望できる日付が表示されない

  • 管理者がシフト枠を作成しているか確認します。
  • そのシフトが開始済みではないか確認します。
  • すでに同じシフトへ希望を出していないか確認します。
  • 別の月を表示していないか確認します。

シフトを承認できない

  • 募集人数に達していないか確認します。
  • 対象のシフト枠が削除されていないか確認します。
  • 承認対象がすでに承認済みではないか確認します。

勤務時間や給与が表示されない

勤務時間と給与は、承認済みかつ終了済みのシフトだけが対象です。承認待ちのシフト、まだ終了していないシフト、削除されたシフトは集計されません。

Notes

現在の実装上の注意

従業員の確認用コードについて

従業員ログインには確認用コード入力画面があります。現在の実装では、確認用コード欄が空でないことを確認し、組織IDとメールアドレスに一致する従業員情報を再確認して従業員ホームへ進みます。

本番運用でメール送信やコード照合を厳密に行う場合は、追加実装が必要です。

CSVエクスポートについて

勤務時間画面にはCSVエクスポートボタンがあります。現在の実装では、ボタンの表示はありますが、CSVファイルを生成してダウンロードする処理は未実装です。

管理者ログアウトについて

管理者ホームにはログアウト導線があります。現在の実装ではログイン画面への遷移が中心です。Firebase Authenticationの明示的なサインアウト処理は、今後の改善対象です。

削除操作について

従業員の削除、シフト枠の削除、組織の削除は、関連データも削除される場合があります。実行前に内容を確認してください。
