README

Chess サービス説明

Chessは、希望シフトの提出からシフト枠の作成、承認、勤務時間と給与目安の確認までを まとめて扱えるシフト管理サービスです。管理者の作業を減らしながら、従業員が働きやすい シフトを作れるようにすることを目指しています。

サービスを見る アピールページへ戻る

Overview

Chessでできること

1

希望シフトを集める

従業員は組織IDとメールアドレスで確認し、管理者が作成したシフト枠に希望を提出できます。

2

承認して確定する

管理者は希望者を見ながら承認できます。募集人数を超えないように承認数も管理されます。

3

働きやすさを考慮する

従業員同士の働きやすさスコアを参考に、おすすめ承認組み合わせを確認できます。

Features

主な機能

管理者向け

  • 管理者アカウントの新規登録、メール確認、ログイン
  • 組織の作成、選択、組織IDの確認、組織削除
  • 従業員の登録、編集、削除
  • シフト枠の作成、編集、削除
  • 提出された希望シフトの確認、承認、承認取り消し
  • 従業員ごとのシフト希望一覧の確認
  • 勤務時間、月間給与目安、年間給与目安の確認
  • 時給、深夜時間帯、深夜倍率の給与設定

従業員向け

  • 組織IDとメールアドレスによる従業員確認
  • 従業員ホーム画面の表示
  • 直近の確定シフト、承認待ち、承認済みシフトの確認
  • 募集シフト枠から希望シフトを提出
  • 複数の希望シフトをまとめて送信
  • 一緒に働きやすさ設定の入力
  • 今月と年間の勤務時間確認
  • 今月と年間の給与目安確認

How To Use

管理者の使い方

01

管理者アカウントを作成する

利用者選択から管理者の新規登録へ進み、メールアドレスとパスワードを登録します。確認メールのリンクを開くとログインできるようになります。

02

組織を作成する

組織名と必要に応じて部署・拠点名を入力します。登録すると6桁の組織IDが発行され、従業員ログインで使えるようになります。

03

従業員を登録する

従業員の氏名、メールアドレス、雇用形態を登録します。登録済み従業員は編集や削除もできます。

04

シフト枠を作成する

日付、開始時刻、終了時刻、募集人数を設定します。作成した枠だけが従業員側の希望シフト入力画面に表示されます。

05

希望シフトを承認する

従業員から提出された希望を確認し、必要な人数分を承認します。承認済みの希望は取り消して承認待ちに戻すこともできます。

06

勤務時間と給与目安を確認する

承認済みで終了したシフトをもとに、従業員ごとの勤務時間と給与目安を確認します。時給や深夜倍率も設定できます。

Employee Flow

従業員の使い方

01

組織IDとメールアドレスで確認する

管理者から共有された6桁の組織IDと、登録済みメールアドレスを入力して従業員ページへ進みます。

02

ホームで状況を確認する

直近の確定シフト、承認待ち、承認済みシフト、勤務時間、給与目安を確認できます。

03

希望シフトを提出する

カレンダーから希望日を選び、募集されているシフト枠を追加してまとめて送信します。

04

働きやすさを入力する

他の従業員との働きやすさをスコアで保存します。この情報は管理者側のおすすめ承認組み合わせに使われます。

Main Appeal

人間関係まで考慮したシフト作成

シフト作成では、単に人数を合わせるだけでなく、「一緒に働きやすいか」という現場の感覚も大切です。 Chessでは、従業員が他の従業員との働きやすさを-5から+5で入力できます。 管理者はそのスコアをもとに、募集人数に対して相性の良い承認候補を確認できます。

+5一緒に働きやすい
0普通
-5一緒に働きにくい
