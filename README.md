# Chess アピールサイト

シフト管理サービス「Chess」の特徴と使い方を紹介する静的サイトです。
現在のChessに合わせて、公平性を考慮したシフト作成、見やすいカレンダー・時間ビュー、管理者と従業員それぞれの操作方法を案内しています。

HTML・CSS・JavaScriptのみで構成されているため、ビルド作業は不要です。

## ページ構成

### `index.html` — アピールページ

Chessの主な価値と画面イメージを簡潔に紹介します。

- 相性・業務スキル・希望時間を考慮したシフト作成
- 勤務機会の偏りを抑える判断材料となる公平性スコア
- カレンダーや時間ビューを使った見やすいシフト管理
- 管理者と従業員それぞれに用意された操作画面
- Chess本体と詳しい使い方への導線

### `guide.html` — 詳しい使い方

管理者・従業員向けの操作を、実際の画面画像とともに説明します。

- ログインと利用者選択
- 従業員・業務スキル・おすすめ計算の設定
- 希望シフトの入力と管理者による代理入力
- おすすめ候補、公平性スコア、シフト承認
- カレンダー、時間ビュー、勤務実績、出力機能
- 従業員のマイカレンダーと勤務履歴

## ページの移動関係

```text
index.html（アピールページ）
├─ サービスを見る ─────> https://chess-nine-xi.vercel.app/
└─ 詳しい使い方 ───────> guide.html
                              ├─ アピールページへ戻る ─> index.html
                              └─ サービスを見る ──────> デプロイ済みChess
```


## 画像について

現在の画面を紹介する画像は、主に`assets/screenshots/current/`に配置しています。
ファイル名は、`admin-time-view.png`や`employee-my-calendar.png`のように、利用者と画面の内容が分かる名前にしています。

画像を差し替える場合は、次の点を確認してください。

- 説明文と画像の内容が一致している
- 個人情報や実在する勤務情報が写っていない
- カードのフレームに合う縦横比と余白になっている
- PC表示とスマートフォン表示の両方で内容を判別できる
- HTML内の画像パスとファイル名が一致している

## ローカルで確認する方法

### ファイルを直接開く

`index.html`をブラウザで開くと、基本的な表示とページ間の移動を確認できます。

### VS Code Live Serverを使う

1. VS Codeでこのフォルダを開きます。
2. `index.html`を右クリックします。
3. **Open with Live Server**を選択します。
4. アピールページから**詳しい使い方**へ移動できることを確認します。

## 更新時の確認項目

- [ ] アピール内容が現在のChessの仕様と一致している
- [ ] `index.html`から`guide.html`へ移動できる
- [ ] 両ページからデプロイ済みChessを開ける
- [ ] すべてのスクリーンショットが表示される
- [ ] 説明文とスクリーンショットの内容が一致している
- [ ] 詳しい使い方の目次と「ページ上部へ戻る」が機能する
- [ ] PC・タブレット・スマートフォンで文字や画像が不自然にはみ出さない
- [ ] キーボード操作でも主要なリンクとメニューを利用できる
