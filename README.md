# Chess アピールサイト

シフト管理サービス「Chess」の特徴と使い方を紹介する静的サイトです。
本採用したアピールページでは、スクロール連動演出を使いながら、相性・業務スキル・公平性を考慮したシフト作成や、カレンダー中心のUIを紹介します。

HTML・CSS・JavaScriptのみで構成されているため、ビルド作業は不要です。

## ページ構成

### `index.html` — アピールページ

通常アクセスで表示される本採用ページです。

- 相性・業務スキル・希望時間を考慮したシフト作成
- 勤務機会の偏りを抑える判断材料となる公平性スコア
- カレンダーや時間ビューを使った見やすいシフト管理
- 管理者と従業員それぞれに用意された操作画面
- Chess本体と詳しい使い方への導線

### `guide.html` — 詳しい使い方

管理者・従業員向けの操作を、実際の画面画像とともに説明します。

### `legacy/` — 旧アピールページ（案0）の保存

案1以前の旧デザインを確認用に保存しています。通常のアピールページからはリンクせず、`legacy/index.html`には`noindex, nofollow`を指定しています。

## ページの移動関係

```text
index.html（本採用アピールページ）
├─ サービスを見る ─────> https://chess-nine-xi.vercel.app/
└─ 詳しい使い方 ───────> guide.html
                              ├─ アピールページへ戻る ─> index.html
                              └─ サービスを見る ──────> デプロイ済みChess
```

`legacy/index.html`へのリンクは通常ページに設置していません。

## ファイル構成

```text
Chess_Appeal_site/
├─ index.html                 # 本採用したスクロール連動アピールページ
├─ appeal.css                 # アピールページの基礎デザイン
├─ appeal-sections.css        # 各章・カード・CTAのデザイン
├─ appeal-responsive.css      # レスポンシブ表示
├─ appeal.js                  # スクロール連動演出
├─ navigation.js              # アピールページのナビゲーション
├─ style.css                  # 共有スタイルと詳しい使い方のデザイン
├─ guide.html                 # 管理者・従業員向けの詳しい使い方
├─ legacy/                    # 案0の保存用ページ（通常導線なし）
├─ README.md
└─ assets/
   └─ screenshots/
      ├─ current/             # 現在のChessを撮影した掲載用画像
      └─ その他の既存画像
```

## ローカルで確認する方法

`index.html`をブラウザで直接開くか、VS CodeのLive Serverで開きます。

## 更新時の確認項目

- [ ] `index.html`から`guide.html`へ移動できる
- [ ] 両ページからデプロイ済みChessを開ける
- [ ] すべてのスクリーンショットが表示される
- [ ] PC・タブレット・スマートフォンで文字や画像が不自然にはみ出さない
- [ ] キーボード操作でも主要なリンクとメニューを利用できる
- [ ] 通常ページから`legacy/`へリンクしていない
